HugoToGithub

使用hugo搭建博客，并在GitHub Pages预览HTML 我使用的编辑器是Visual Studio Code，命令行工具是Bash。 从hugo的 GitHub Releases下载相应的版本 添加hugo.exe的路径到Path，这命令行工具输入hugo version验证 建立一个网站，名为blog hugo new site blog 下载主题 cd blog git init git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke 然后，将主题添加到站点配置中： echo 'theme = "ananke"' >> config.toml 添加文章 添加一个测试文件test.md hugo new posts/test.md 编辑test.md，原来的内容不要删除，编写完内容修改draft为false。 6. 启动hugo服务器 hugo server -D 按住ctrl单击运行完的链接，例如http://localhost:xxxx/，之后你就能在浏览器查看test.md的效果。博客已完成，接下来上传到GitHub。 建立静态页面 hugo -D 执行完，项目会增加一个public文件夹，这个就是要上传到GitHub仓库的。 8. 初始化public仓库 首先在blog项目下创建.gitignore文件，里面添加/public/，然后直接下面命令. cd public git init 添加到本地仓库 git add.
